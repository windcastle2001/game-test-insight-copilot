02 · User Flow사용자 흐름

평가자/실사용자가 도구에 진입한 시점부터 결과 보고서를 다운로드하기까지의 전 단계.

1
입장 & 컨텍스트 파악 헤더와 입력 패널이 화면에 노출됨. 제품명·태그를 보고 "AI 의사결정 도구"임을 즉시 인식.
2
데이터 입력 — 3가지 옵션 중 선택

옵션 A · Quick Start — 샘플 게임 3종 카드 클릭 (SCALE / ITERATE / KILL) → 라이브 지표 + 동향 자동 채워짐

옵션 B · 직접 업로드 — raw 지표 CSV + 동향 CSV 각각 업로드. 필요 시 지표 컬럼 추가/삭제 후 빈 양식 다운로드 → 채워서 재업로드

옵션 C · 혼합 — 샘플로 시작 후 지표나 동향 데이터를 일부만 직접 교체

3
분석 시작 "Gemini로 분석 생성" 버튼 클릭 → 진행 모달이 나타나고 프로그레스 바가 실시간 업데이트.

5% — Raw data 검산

15~65% — 동향 원문 청크별 Gemini 분석 (N/총개수)

68% — 동향 분석 결과 취합

72% — Gemini 최종 의사결정 생성

95% — 실행 계획 정리

100% — 결과 노출

4
결과 확인 결과 영역으로 자동 스크롤. Decision → KPI 진단 → 차트 → AI 인사이트 → 실험 계획 → 회의 요약 순으로 보고.
5
보고 자료 추출

PDF 다운로드 — 보라색 버튼 클릭 → A4 다중 페이지 PDF로 결과 전체 저장

회의 요약 복사 — 한·영 요약문을 클립보드에 복사 후 슬랙/노션에 붙여넣기

재분석 — 지표나 동향을 수정하고 분석 버튼 재클릭하면 결과 갱신
